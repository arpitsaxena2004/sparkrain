About JalNidhi AI

JalNidhi AI is an AI-powered rooftop rainwater harvesting assistant that estimates water capture potential, predicts installation cost, and suggests site suitability for Indian districts.

Try the Calculator View Dashboard

Water Potential

Estimates the rainwater you can capture based on rooftop area, annual rainfall, and runoff factor.

AI Suitability

Uses a clustering model to categorize your site suitability using rainfall and soil type signals.

Cost Prediction

Predicts approximate installation cost using a trained model with inputs like suitability, tank size, and area.

How it works

  1. District validation: only Indian districts present in the training dataset are accepted (with typo suggestions).
  2. Weather snapshot: pulls current temperature, humidity, and rainfall (if available) from OpenWeatherMap for context.
  3. ML inference: predicts suitability and cost using pre-trained models stored in ml_models/.
  4. Water calculation: estimates captured water using rooftop area × rainfall × runoff factor.

Note

Predictions are estimates to support planning and awareness. For final design, consult local guidelines and a qualified professional.

Key inputs

  • Rooftop area (sq meters)
  • Annual rainfall (mm)
  • Soil type (Loamy/Clay/Sandy/Rocky)
  • Preferred tank capacity (liters)

Built with

  • Django web app with Tailwind UI
  • Scikit-learn models (KMeans + RandomForest)
  • Groq LLM API (LLaMA 3.1 70B) for AI Chatbot
  • OpenWeatherMap API for live weather context
