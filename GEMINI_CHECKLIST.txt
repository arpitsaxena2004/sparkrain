STEP 3: ADD TO .ENV FILE

[ ] Open .env file in editor
[ ] Find line: GEMINI_API_KEY=
[ ] Paste your key after the = sign
[ ] Should look like: GEMINI_API_KEY=AIzaSy...
[ ] NO spaces, NO quotes
[ ] Save file (Ctrl+S or Cmd+S)

Example:

Problem: "Module not found"
Solution: pip install google-generativeai

Problem: "API key not configured"
Solution: 
  1. Check .env file has: GEMINI_API_KEY=AIzaSy...
  2. No spaces: GEMINI_API_KEY=key (not = key)
  3. No quotes: GEMINI_API_KEY=key (not "key")
  4. Save file and restart server

Problem: "API key invalid"
Solution:
  1. Go to https://makersuite.google.com/app/apikey
  2. Delete old key
  3. Create new key
  4. Copy and paste to .env
  5. Restart server

Problem: Still generic responses
Solution:
  1. Check console for errors
  2. Verify internet connection
  3. Check API quota not exceeded
  4. Try new API key
